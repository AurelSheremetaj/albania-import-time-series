# Time Series Analysis of Albania's Quarterly Import Price Index

## Description
This project analyzes the quarterly import price index of Albania using time series methods in R. The study focuses on identifying trend and seasonal components, exploring data behavior, and building forecasting models.

## Methods Used
- Data visualization
- Time series decomposition
- Trend and seasonality analysis
- ARIMA modeling
- Forecasting

## Tools
- R
- tidyverse
- ggplot2
- forecast
- readxl
